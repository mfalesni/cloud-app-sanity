test_audreyvm11remoteyuminstall Module¶

This module contains tests which are supposed to test remote control Katello -> guest computer

test_audreyvm11remoteyuminstall.test_install_packages_remote(audreyvars, system_uuid, tunnel_requested)[source]¶

Installs packages specified in YUM_REMOTE_INSTALL into this system via remote request through Katello server to check whether there aren’t any issues.

Parameters:
  • audreyvars (dict) – Audrey environemnt variables
  • system_uuid (str) – This system’s unique ID for Katello
  • tunnel_requested (bool) – Whether was tunnel requested or not
Raises :

pytest.Skipped, pytest.Failed
